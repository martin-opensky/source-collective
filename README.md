# To Get started

Install dependencies: _yarn_

# Dev

_yarn dev_

Will start Next.js server and Tauri app

# Build

_yarn build_

_yarn build --debug_ _to include dev tools in app_

Will build Next.js static app and Tauri app (for Mac)
